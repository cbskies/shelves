CSS3/Javascript Shelves Usage

Thanks for downloading the shelves app!. This page demonstrates the usage for your downloaded shelves design.

All you need to create a shelve system on a web page is the following:

  • Include the shelves.css style sheet
  • Include the shelves.js javascript file
  • Include the jquery-1.11.0.min.js javascript file
  • Use the div structure to wrap your desired shelf content as it is shown below and demonstrated on this page
